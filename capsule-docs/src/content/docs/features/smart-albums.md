---
title: Smart Albums
description: Automatically generated albums
---

Given standard and AI-generated metadata, there are custom, live-updated albums generated:

| Type            | Description                           |
| --------------- | ------------------------------------- |
| People          | One album per person                  |
| Trips           | GPS+temporal clustering               |
| Pets            | Pet detection + breed ID              |
| Food            | Food recognition                      |
| Nature/wildlife | Scene + animal classification         |
| Best-of-year    | Aesthetic Quality + temporal grouping |

Note: These albums are dynamically generated so you can equivalently do static queries to get the same or combined results.
